# `@dongmingchao/prettier-config`

> My personal [Prettier](https://prettier.io) config.

## Usage

**Install**:

```bash
$ yarn add --dev @dongmingchao/prettier-config
$ npm i -D @dongmingchao/prettier-config
```

**Edit `package.json`**:

```jsonc
{
  // ...
  "prettier": "@dongmingchao/prettier-config"
}
```
